Задание для желающих получить экзамен по веб-языкам

Вам нужно создать приложение на Express, которое поддерживает сессии

Список записей вида login:password находится в файле users.json

Путь /login должен вести к форме для ввода логина и пароля

В случае успешного ввода сессия начинается.

Для того, кто успешно вошёл, доступен путь /profile где он видит свой логин и гиперссылку на /logout

При заходе на /logout сессия уничтожается.

При заходе на "приватный" путь вроде /profile без предварительной аутентификации нас должны перекидывать на /login

А после успешного логина возвращать туда, куда мы хотели пройти.

Пример.

По адресу http://exam2.kodaktor.ru/users вы видите список пользователей

По адресу http://exam2.kodaktor.ru/users/profile - закрытый путь, который перекидывает на /users/login с формой

Обратите внимание: форма делает запрос методом POST к пути /users/login/process

Ещё пример закрытого пути: http://exam2.kodaktor.ru/users/raw

Закрытые пути "перекрываются" функцией middleware, которая в случае действующей сессии делает next(), а в противном случае 
редиректит на путь с формой логина.

Вам нужно также использовать шаблонизатор и BootStrap для красивости формы и других страниц.

Исходные коды должны быть в репозитории. В файле README.MD кратко опишите, какой механизм работы с сессиями вы использовали 
и какими примерно были ваши шаги.
